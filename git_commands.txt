1.安装Git
git config --global user.name "yourname"

2.创建版本库
mkdir learngit
cd learngit
pwd					pwd命令用于显示当前目录
git init				把目录变成Git可以管理的仓库
git add readme.txt
git commit -m "wrote a readme file"

3.时光穿梭机
git status
git diff readme.txt
3.1版本回退
git log
git log --pretty=oneline
git reset --hard HEAD^(HEAD~100)
cat readme.txt				打开readme.txt
git reset --hard 1094a
git reflog
3.2工作区和缓存区
Working Directory Repository stage(index) master HEAD
3.3管理修改
git diff HEAD -- readme.txt
3.4撤销修改
git checkout -- readme.txt		尚未add
git reset HEAD readme.txt		add之后commit之前，变回上一步
git reset --hard HEAD^(HEAD~100)	commit之后，push之前
3.5删除文件
rm test.txt
git rm test.txt
git commit -m "remove test.txt"

5.分支管理
5.1创建与合并分支
master指向某个提交 HEAD指向当前分支
git checkout -b dev
git branch dev
git checkout dev
git branch
git merge dev
git branch -d dev
5.2解决冲突
git merge feature1
git add readme.txt
git log --graph --pretty=oneline --abbrev-commit	--graph命令可以看到分支合并图
5.3分支管理策略
git merge --no-ff -m "merge with no-ff" dev
分支策略：master发布版本，dev工作，各人在dev的分支上干活
5.4bug分支
git stash
git stash list
git stash apply + git stash drop
git stash pop
git stash apply stash@{0}
5.5feature分支
git branch -D feature-vulcan
5.5多人协作
git remote
git remote -v
git push origin master
git push origin dev
master和dev分支要时刻与远程同步
git clone git@github.com:michaelliao/learngit.git	只能看到master需再checkout
git checkout -b dev origin/dev
git pull						本地dev和远程dev未连接
git branch --set-upstream-to=origin/dev dev
